Lesson 0 - Downloading VS code

What is VS code?

Visual Studio Code (VS Code) is a free, lightweight, and powerful code editor made by Microsoft. You’ll use it throughout this course to write, edit, and run Python code.

Installing VS code on Windows

To install VS code on your computer, perform the following tasks:


  1. Go to code.visualstudio.com


  2. Click Windows (User Installer)


  3. Run the installer and:
    • Accept the licence agreements
    • Check "Add to PATH"
    • Finish Installation

Installing VS code on MacOS

To install VS code on your Mac, perform the following tasks:


  1. Go to code.visualstudio.com


  2. Download the MacOS Universal version


  3. Open the .zip and drag Visual Studio Code into Your Applications folder

Installing VS code on Linux

To download VS code on Linux, you have to run a command based on the distro of linux you are using:


  • Debarian/Ubuntu (.deb)

    bash

    $ sudo apt install ./<downloaded-file>.deb
  • Fedora (.fed)

    bash

    $ sudo rpm -i <downloaded-file>.rpm
  • Snap Store (Any distro with Snap)

    bash

    $ sudo snap install code --classic
